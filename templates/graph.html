I Gotchu!

To do any kind of selection on the graph, you must activate "Selection Mode" by clicking on the button at the top right corner of the page.

For Nodes:

  • Select a node to edit with Shift + E.
  • The node's key/value and name will appear in the input boxes to the left.
  • Do not edit the node's name to update the value of that node.

For Edges:

  • Select a source node with Shift + F.
  • Select a target node with Shift + T.

For Delete:

  • Select a node with Shift + D.
